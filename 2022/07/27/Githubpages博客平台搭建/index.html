Githubpages博客平台搭建

发表于 2022-07-27 更新于 2022-05-28

静态Blog框架的选择（Hexo & Jekyll）

首先说一下我的选择——Hexo。

  • Jekyll基于Ruby实现，需要搭建Ruby环境；而Hexo是基于NodeJs实现的，在windows上安装NodeJs比较简单；
  • Hexo中的Next主题，经典中的经典；
  • 两者都支持Markdown语法（这非常重要）。

为什么要搭建Blog？

我对于这个问题的回答是分享，分享自己的知识，分享自己的思考，分享自己的想法。对于计算机的学习，做笔记是一件很重要的事情，把他们写在博客里，可以让大家看到，帮助到大家，有错误的地方网友们可以指出并提出自己的修改意见，让自己成为更好。我想这是一件很快乐且很重要的事情，且坚持做一件事不是很酷嘛！！！

如何搭建Github pages博客

1.注册github账户

首先你得有一个Github账号，github官网：https://github.com/ 。

登入自己的账号后，

直接点击creat repository就可以了，这时候输入自己的用户名.github.io就可以看到自己的一个简单的博客，是readme.md文件中的内容。例如，我的Yunfengsun.github.io 。

2.安装 Node.js

网址：https://nodejs.org/en/

点击16.14.2 LTS下载，这个是稳定版本的，下载完成之后点击安装。

安装完成之后打开CMD（win + R）输入“node -v”，得到“v16.14.2”；输入“npm -v”，得到“8.5.0”，安装成功。

3.安装git

网址：https://git-scm.com/

点击download，点击windows，下载完成之后点击安装。

安装完成之后后打开CMD（win + R）输入“git —version”，得到“git version 2.35.1.windows.2”，安装成功。

4.安装hexo

打开CMD（win + R）输入“npm install -g hexo-cli”，回车即开始安装，输入“hexo -v”，得到“hexo-cli: 4.3.0”等一连串版本号，安装成功。

5.搭建本地博客

创建一个文件夹，命名随意（例Blog），右键文件夹或键入文件夹右键，选择Git Bash Here。

输入“hexo init”将文件夹初始化为一个博客文件夹，再输入“npm install”安装依赖；

输入“hexo g”生成网页，初始网页为hexo自带的hello world的博客。

输入“hexo s”将生成的网页放在本地服务器上，打开浏览器输入：http://localhost:4000/ ，即可看到自己的网页。ctrl + C关闭本地服务。

6.发布一篇自己的博客

在创建的文件夹下打开source/_posts，在里面新建markdown文件，例如（first.md），后输入“hexo g”，再输入“hexo s”，打开浏览器输入“http://localhost:4000/ ”，就可以看到自己创建的内容了。
